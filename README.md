# aether-veil-starter

Production-ready starter for a luminous 3D gacha RPG targeting Vercel deployment.

## Stack

- Vite 6 + React 19 + TypeScript
- Babylon.js 8.x scene bootstrap
- Tailwind CSS + shadcn/ui baseline config
- PWA via `vite-plugin-pwa`
- Supabase client scaffold
- Zustand game store scaffold

## Quick start

```bash
npm install
npm run dev
```

## Environment

Create `.env.local`:

```bash
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```
